CROPS

CROPS_288X180N.ext

 Uses also land mask. Denis Gueyffier has created this file using Reto Ruedy's data & programs in dirac:/archive/u/rruedy/GISS/crops.tar original data defined on a 1/2 x 1/2 degree grid. Original dataset from Ramankutti and Foley covering the period 1700-1992 http://www.earthsystematlas.org/explanations/crop_data/crop_data_technical.html

288x180 version of conversion program at /discover/nobackup/dgueyffi/create_crops_inputfile/crops_288x180.f
